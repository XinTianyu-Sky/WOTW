# 风尘叹 (Whispers of the Wanderer) 版本记录

## v0.1.0 — MVP 核心循环 (2026-05-17)

### 新增
- 主菜单：新游戏、读档、退出
- 角色创建：6 种出身选择、六维属性分配、姓名验证
- 世界探索：TileMap 地图（40×30 洛阳郊外）、WASD 移动、Shift 冲刺、镜头跟随
- HUD：生命/内力条、等级、时间、天气、底部功能栏
- NPC 交互：按 E 键触发对话（分支选项、好感度影响、任务触发）
- 野外遇敌：草地/树林行走随机遇敌、累积距离检测
- 战斗系统：六边形网格回合制、五行克制、自动战斗（MVP）、战败/胜利判定
- 存档系统：F5 手动存档、HUD 存档按钮、进世界自动存档、遇敌前自动存档、主菜单读档
- 背包系统：6 列网格、物品详情、稀有度颜色标识
- 角色面板：六维属性、战斗属性表
- 武学面板：外功/内功/轻功三页切换、招式详情
- 任务系统：进行中/已完成追踪、对话触发接受、完成奖励发放
- 数据结构：16 个 JSON 数据文件（武学/物品/角色/任务/世界）
- 5 个 Autoload 单例：GameManager、DataManager、EventBus、SaveManager、NotificationManager

### 已知限制
- 战斗为自动模式，无手动技能选择 UI
- 角色/NPC/地图使用占位色块精灵，无正式美术素材
- TileMap 碰撞仅在阻塞瓦片生效，无 Y 排序
- 对话条件系统未完全实现（好感度/物品检查）
- 无音效/音乐
- 仅在 PC 端测试，移动端虚拟摇杆已置入但隐藏
