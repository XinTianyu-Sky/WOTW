# 风尘叹 (Whispers of the Wanderer)

> 一款以中国传统武侠江湖为背景的2D开放世界角色扮演游戏。

## 简介

《风尘叹》是一款类似《烟雨江湖》的武侠开放世界RPG。玩家将扮演一名身世成谜的江湖游子，在风云变幻的虚构江湖世界中自由探索、拜师学艺、行侠仗义，最终揭开身世之谜，走出属于自己的江湖之路。

## 核心特色

- **自由探索**：非线性的开放世界，六大区域（中原、蜀中、北冥、东海、南疆、西域），无强制路线
- **深度武学**：外功（招式）+ 内功（心法）+ 轻功（身法）三维武学体系，五行内力克制，经脉系统
- **江湖人情**：NPC日程+好感度+记忆系统，玩家的行为影响声望和江湖格局
- **策略战斗**：六边形网格回合制，招式连击链，异常状态，Boss阶段AI
- **丰富养成**：六大门派+三大世家，8名可招募同伴，锻造/炼丹/裁缝/烹饪制造体系

## 技术栈

- **引擎**：Godot 4.x（推荐）
- **语言**：GDScript + C++（GDExtension）
- **美术**：2D像素风 + 中国水墨风UI
- **平台**：iOS / Android（首发）→ Steam（第二阶段）

## 开发规范

- **GDScript缩进**：必须使用Tab缩进。Godot编辑器会自动将空格转为Tab。
